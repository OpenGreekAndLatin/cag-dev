Welcome to cag-dev!

About

This repository hosts digital versions of the Commentaria in Aristotelem Graeca (CAG).

These XML files were produced by data entry company DDD, who OCR'd, corrected and encoded each volume of CAG in accordance with the latest EpiDoc standards. While adequate, more can and should be done to improve the XML (the -dev(elopment) suffix indicates the repository is open to improvement).
